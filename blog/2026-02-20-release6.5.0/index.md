# Release 6.5.0

:::info
Release 6.5.0 is out this week with improved user experience and home page enhancements!
:::

## What's New

**SF Explorer 6.5.0** brings a refreshed user experience with a comprehensive review and improvement of the home page:

### üè† Enhanced Home Page Experience

We've reviewed and improved the home page to provide a better user experience:

- **Improved Navigation**: Streamlined access to key features and quick links
- **Better Visual Layout**: Enhanced organization of content and features
- **User-Friendly Design**: More intuitive interface for both new and returning users
- **Optimized Content**: Clearer presentation of key features and capabilities

The home page now offers a more welcoming and efficient entry point to SF Explorer, making it easier for users to discover and access the tools they need.

## What and Why

This release focuses on improving the first impression and usability of SF Explorer. By reviewing and enhancing the home page, we aim to:

- **Reduce Learning Curve**: Help new users understand SF Explorer's capabilities faster
- **Improve Productivity**: Make it easier for users to find and access frequently used features
- **Enhance User Satisfaction**: Create a more polished and professional experience

We believe that a great first impression starts with a well-designed home page, and version 6.5.0 delivers exactly that.
